## Doxygen vs Swagger
### Doxygen 
- 목적 : 코드 문서화
- 기능
  - 코드 주석 (주석 분석하여 API 문서, 클래스 다이어그램 등을 생성)
  - 문서 포맷
  - 호환성
- 용도 : 소스 코드의 기능, 클래스, 메서드 등을 설명하는 기술 문서를 자동으로 생성

### Swagger
- 목적 : API 문서화
- 기능
  - API 명세 (Request, Response 등)
  - 문서화 (APIs)
  - 자동 생성(클라이언트 라이브러리 등)
- 용도 : RESTful API 구조 정의, 시각화
